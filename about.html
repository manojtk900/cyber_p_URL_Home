About This Project

ASTRAL PROTECTION is a phishing website detection system built using a trained Machine Learning model. Users paste URLs and the model predicts whether a website is SAFE or PHISHING. The UI also features a helpful AI ChatBot that answers questions about phishing and how the detector works.
